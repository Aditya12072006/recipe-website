## Project Overview
I built this multi-page "Delicious Recipes" website for my Web Development assignment. The goal was to create a functional, responsive, and visually clean site using only **HTML and CSS**. As per the assignment rules, no JavaScript was used for any part of the site.

The website features a homepage and five detailed recipe pages, focusing on clean navigation and a layout that works across different screen sizes.

## Project Features
* **HTML Structure:** I used semantic HTML to keep the code organized and easy to read.
* **Recipe Database:** Includes five different recipes: Pasta, Biryani, Chocolate Cake, Salad, and Dosa. Each page has ingredients, prep time, difficulty levels, and step-by-step instructions.
* **CSS Layouts:**
    * **Flexbox:** Used for the header and navigation bar to keep everything aligned.
    * **CSS Grid:** Used on the homepage to create a clean, card-based layout for the featured recipes.
    * **Box Model:** Careful use of margins, padding, and borders to ensure the content has enough breathing room.
* **Responsive Design:** I included media queries in the CSS so the website scales down nicely for mobile devices. On smaller screens, the recipe grid and navigation links stack vertically.
* **User Interface:** Added hover effects on links and buttons to make the site feel interactive without needing scripts.
